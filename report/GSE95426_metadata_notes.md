## Notes

This dataset is used as an initial pilot dataset rather than as the sole discovery dataset. Its main role is to test whether the literature-curated POCD-related modules can be scored and evaluated in a public hippocampal expression dataset.

The project will avoid making strong causal claims from this dataset alone. Future steps will include cross-dataset validation using additional POCD/PND-related datasets.
